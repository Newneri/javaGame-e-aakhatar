# Java Console Game

A console-based Java game featuring level generation and player movement.

## Description
This project implements a game with grid-based maps where a player can move around. It handles collision detection, score tracking, and map rendering.

## Project Structure
- **src/generation/Level.java**: Core game logic, handles map loading, rendering, and player movement.
- **src/generation/Player.java**: Player class managing position, score, and identity.

## Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher.

### Compilation
To compile the project, navigate to the `src` folder:

```bash
cd src
javac generation/*.java
```

### Running the Game
To run the game (executing the Level class):

```bash
java generation.Level
```
